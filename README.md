# gFreqReg

This repo contains the source code for regression method in the graph frequency domain.

## Description

- Code
  - `utils.R` is a code for functions used for graph frequency-domain regression.
  - `main.R` is codes for simulation study and real data application on G20 network data.

- Data
  - `tradingnw.RData` contains the G20 trade network data for 2023.
  - `BACI_HS92_V202501` (available at [http://www.cepii.fr/anglaisgraph/bdd/baci.htm](http://www.cepii.fr/anglaisgraph/bdd/baci.htm)) and `economic` contain information on world trade and economic indicators.

## Code overview
We present a regression method in the graph frequency domain.
